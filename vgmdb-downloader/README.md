# vgmdb-image-downloader

A downloader for [VGMdb](https://vgmdb.net). Userscript that adds a "Download All Images" button to album pages for batch downloading cover art.

## Usage

1. Install a userscript manager (e.g. [Tampermonkey](https://www.tampermonkey.net/))
2. Create a new userscript and paste the contents of `src/downloader.js`
3. Visit any album page on [VGMdb](https://vgmdb.net)
4. Click the **Download All Images** button above the cover gallery
