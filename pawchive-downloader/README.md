# pawchive-downloader (archived)

> **⚠️ Archived — no longer maintained.**
> Development has moved to **pawchive-downloader-v2** (a sibling folder), a
> behavior-preserving rewrite with a cleaner CLI and the same download engine.
> Your artist data and cache were migrated there; this copy is kept for
> reference only and receives no further updates.

Downloads creator posts and attachments from [Pawchive](https://pawchive.pw) — an
interactive shell that tracks creators, caches their post lists, and fetches new
files incrementally, concurrently across creators, posts and files.

The engine here is unchanged and still runnable (`python main.py`), but all new
work happens in v2. See `../pawchive-downloader-v2/README.md`.
